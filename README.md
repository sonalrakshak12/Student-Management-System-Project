# Student Management System

A simple Java console-based application that helps manage student records. This project is designed for beginners to understand Object-Oriented Programming (OOP) concepts and CRUD (Create, Read, Update, Delete) operations in Java.

## Features

- Add Student
- View All Students
- Search Student by ID
- Delete Student by ID
- Menu-driven Console Interface

## Technologies Used

- Java
- ArrayList
- Object-Oriented Programming (OOP)
